종횡비 계산기 (Aspect Ratio)

이미지, 동영상, 레이아웃의 가로 세로 비율을 계산하고 최적의 크기를 제안합니다.

비율 입력

계산된 비율: 16:9 (1.78:1)

디자이너 추천 규격

실시간 화면 매칭

1920 x 1080 16:9

PX ↔ REM 변환기

픽셀(Pixel) 값과 상대 단위인 렘(Rem) 값을 상호 변환하고, 최적화된 반응형 폰트 코드를 생성합니다.

변환기

px
px
rem

CSS 변수 코드 자동 생성

/* CSS Variables Output */
:root {
  --font-size-custom: 1rem; /* 16px */
}

빠른 참조 테이블

픽셀 (PX) 렘 (REM) 용도 추천 복사

WCAG 색상 대비 검사기 (Contrast Checker)

전경 글꼴 색상과 배경 색상의 명도 대비를 계산하여 웹 콘텐츠 접근성 지침(WCAG 2.1) 충족 여부를 검사합니다.

색상 설정

계산된 명도 대비
5.41 : 1
AA (보통 텍스트) 통과
AA (큰 텍스트) 통과
AAA (보통 텍스트) 미달
AAA (큰 텍스트) 통과

실시간 디자인 대비 미리보기

AA 등급 대비 기준 (본문 글 기준 4.5:1 이상)

우리는 디자이너와 퍼블리셔가 디자인 시안을 설계하고, 접근성 가이드라인을 손쉽게 맞출 수 있도록 이 도구를 제작했습니다.

큰 제목 텍스트 (18pt / 24px 이상 볼드체 - 3:1 이상)

가독성 테스트를 위한 샘플 텍스트입니다. (Bold)

가독성 최적화 팁

선택된 색상 조화는 명도 대비율이 양호하여 웹 콘텐츠 접근성 지침(WCAG) AA 기준을 완전하게 만족합니다.

그리드 레이아웃 계산기 (CSS Grid Calculator)

컨테이너의 전체 가로폭, 그리드 컬럼 수, 거터(Gutter), 여백(Margin)을 설정하여 개별 컬럼폭을 계산하고 그리드 코드를 생성합니다.

그리드 변수 설정

단일 컬럼 너비 (Column Width): 73.33 px
총 컨텐츠 가로폭 (Contents Width): 1136 px

CSS Grid 코드 자동 생성

/* CSS Grid Generated */
.grid-container {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  gap: 24px;
  max-width: 1200px;
  padding: 0 32px;
}

그리드 레이아웃 시각화

바깥쪽 패딩(여백)과 안쪽 사이 공백(거터)이 들어간 상태의 CSS 컬럼 배치 구조를 1:1 비율로 축소 표현한 미리보기입니다.
